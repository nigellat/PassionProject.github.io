Scratch

Scratch is an online platform for coding with block code. It is perfect for beginner coders as the use of block code is easier to start out with while still letting you create complex code. Typically Scratch is used to make games. Some schools teach younger kids how to use this program to start learning code for the very first time and to spark interest in coding early on. Despite that, people of any age can use Scratch and since its easily accessible there is already a large community of people who like to create code and games on Scratch. If you want to start coding with Scratch all you need to do is go to Scratch.mit.edu and create an account.
